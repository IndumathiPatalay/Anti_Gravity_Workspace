# Anti-Gravity Workspace

Welcome to the Anti-Gravity Workspace repository. This monorepo workspace houses multiple projects, each focused on developing advanced applications.

## 📁 Repository Projects

### 1. 🧠 [Dementia ASR Screening](./Dementia_ASR_Screening)
A full-stack cognitive screening application that leverages **Automatic Speech Recognition (ASR)** to transcribe and analyze verbal responses, aiding in the early detection of cognitive decline and dementia.
*   **Frontend**: React 19, TypeScript, Vite, Vanilla CSS.
*   **Backend**: Node.js, Express, SQLite3, Multer.
*   👉 **[Dementia ASR Screening Documentation](./Dementia_ASR_Screening/README.md)**

### 2. 📋 [ProcureEase](./ProcureEase)
A procurement management platform designed to streamline corporate purchase requests, workflows, and supply tracking.
